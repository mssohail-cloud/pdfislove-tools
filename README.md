# PDFIsLove Tools

A growing collection of free, open-source PDF utilities and examples for developers.

🌐 Website: https://pdfislove.net

## About

PDFIsLove is a free online platform that helps users work with PDF documents efficiently. This repository contains open-source utilities, code samples, and developer resources related to PDF processing.

## Features

- Merge PDF
- Split PDF
- Compress PDF
- Rotate PDF
- Delete PDF Pages
- Extract PDF Pages
- Organize PDF
- Watermark PDF
- Protect PDF
- Unlock PDF
- PDF Metadata Reader
- OCR Examples
- Image to PDF
- PDF to Image
- Word to PDF
- Excel to PDF
- PowerPoint to PDF
